Strings

  • base/concat.h

Provides base::concat(...) and base::concat_to(&str, ...) functions for building strings through concatenation of pieces. Also defines an interface for objects to declare themselves stringable.

Results

  • base/result.h
  • base/result_testing.h

Provides a base::Result value type representing success or failure of an operation, plus macros for checking such values in tests.

Logging

  • base/logging.h
  • base/debug.h

Provides macros for generating log messages, including logged assertions. Also provides an API for consuming/intercepting logs, and provides functions for checking and changing the global debugging mode.

Date/Time

  • base/duration.h
  • base/time.h
  • base/clock.h
  • base/clockfake.h
  • base/stopwatch.h

Provides a number of classes and value types for representing and manipulating time:

  • base::Duration represents a span of time.
  • base::Time and base::MonotonicTime represent instants of time.
  • base::now() and base::monotonic_now() obtain time values.
  • Alternatively, the base::Clock and base::MonotonicClock holder classes can be used directly.
  • Custom clock implementations can be created via the base::ClockImpl and base::MonotonicClockImpl abstract base classes.
  • base::Stopwatch can be used to measure spans of elapsed time.

Files (low level)

  • base/fd.h

Provides base::FDHolder (a wrapper for file descriptors) and base::FD (a smart pointer for base::FDHolder). Wrapping a file descriptor ensures that it gets closed when no longer in use, and also protects against the recycling of file descriptor numbers.

Miscellaneous

  • base/backport.h

Provides backports of C++14 and/or C++17 features.

Provides base::backport::make_unique(), for constructing objects wrapped in std::unique_ptr.

Provides base::backport::integer_sequence, base::backport::make_integer_sequence, base::backport::index_sequence, and base::backport::make_index_sequence, for indexing tuples.

  • base/cleanup.h

Provides base::Cleanup, a RAII class to call a functor upon leaving a scope.

  • base/mutex.h

Provides mutexes, locks, and other concurrency tools.

  • base/token.h

Provides base::token_t, a value type representing a unique opaque token.
