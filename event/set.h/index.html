RC. event::Set is a value type that acts as a container for a fixed list of one-bit flags. Its primary purpose is to indicate which events an event handler is interested in, and to inform the event handler of which events actually occurred. These flags are:

  • readable: There is data available to read on a pipe or socket.
  • writable: There is room in the buffer to write data to a pipe or socket.
  • priority: There is high-priority data available to read on a socket.
  • hangup: The remote end of a socket has closed.
  • error: An error has occurred on a socket.
  • signal: The process has received a signal.
  • timer: A timer has elapsed.
  • generic: A "generic event" (see event/manager.h) has been triggered.
